# DAD's Games

A collection of browser-based games built with vanilla HTML, CSS, and JavaScript — no dependencies or build step required.

**Live site:** https://ebfeinstein.github.io/games/

## Games

### Hangman ([hangman.html](https://ebfeinstein.github.io/games/hangman.html))
Guess the hidden word one letter at a time before the hangman is complete. Built-in word list, on-screen keyboard, and physical keyboard support.

### Tic-Tac-Toe ([tictactoe.html](https://ebfeinstein.github.io/games/tictactoe.html))
Classic 3×3 grid game. Player vs Player or Player vs unbeatable minimax AI. Tracks score across rounds.

### 3D Tic-Tac-Toe ([tictactoe3d.html](https://ebfeinstein.github.io/games/tictactoe3d.html))
Four-in-a-row on a 4×4×4 cube — 76 winning lines across rows, columns, and diagonals in any direction. Player vs Player or Player vs AI. The AI is intentionally beatable (player wins roughly 75% of the time).

## Running locally

Just open any `.html` file directly in a browser — no server needed.

```
open index.html
```

## Tech

Single-file HTML apps — CSS, markup, and JavaScript all inline. No frameworks, no dependencies, no bundler.
