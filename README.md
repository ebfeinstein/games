# Browser Games

A collection of browser-based games built with vanilla HTML, CSS, and JavaScript — no dependencies or build step required.

**Live site:** https://ebfeinstein.github.io/games/

## Games

### Hangman
Guess the hidden word one letter at a time before the hangman is complete. Features a built-in word list and on-screen keyboard.

### Tic-Tac-Toe
Classic two-player 3×3 grid game. First to get three in a row wins.

### 3D Tic-Tac-Toe
A four-in-a-row variant played on a 4×4×4 cube. Win by connecting four cells in any row, column, or diagonal across any plane or through the cube.

## Running locally

Just open any `.html` file directly in a browser — no server needed.

```
open index.html
```
